# serve-pkg

[![Build Status][build-badge]][build-status]

[serve](https://github.com/zeit/serve) + [pkg](https://github.com/zeit/pkg)

## Usage

See [docs of serve](https://github.com/zeit/serve)

## Download

Download from [releases](https://github.com/airt/serve-pkg/releases) (built by [travis-ci](https://docs.travis-ci.com))

## How to Build

Run the `build` command with npm or yarn: `npm i && npm run build`

The generated files will be placed in the `out` directory
